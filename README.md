# To-do-list-django
To-Do list application made in Django framework using sqllite3 database.
It will help you to maintain a list of your daily tasks.

## Features :

* Add new task
* Update your task and Mark task as completed / uncompleted
* Delete single task
## Technologies used:

* Django 3.0
* Pyhton
* Sqlite3
* Html ; CSS ;Jinja2
